Privacy Policy for Picture Overlay

Last updated: January 4, 2023

While this application accesses data such as photos from the user(s) device or the internet, this application does not collect/store any personal information it accesses on behalf of the user(s) or at the user(s) request. Data accessed while using this application may be stored locally on the user(s) device within this applications storage. However, upon uninstalling this application, all locally stored data are permanently removed from this applications storage.

For the purposes of this Privacy Policy: • User(s) means the individual or individuals accessing or using the application, or the company, or other legal entity on behalf of which such individual is accessing or using the application, as applicable. • Data means photos accessed by the user(s) for use within this application. • This Device means the device where this application is installed and being used.

Collection and Usage of Personal Data

Usage Data

No usage data is collected while using this application.

Retention of Personal Data

No personal data is collected or retained outside this applications storage without the user(s) permission.

Transfer of Personal Data

No personal data is ever transferred outside of this device by this application for any reason.

Deletion of Data

All personal data, such as photos stored within this applications storage, are deleted from this applications storage when this application is uninstalled.
